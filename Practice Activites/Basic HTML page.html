CS2350: Client-side Web Development

HTML5

HTML stands for "Hypertext Markup Language" used for structuring and presenting web pages and is a language used for the world wide web. HTML5 is the fifth and last major HTML version recommend to use.

CSS3

CSS stands for "Cascading Style Sheets" used for visual presentation of HTML or other markup languages. It can control the colors, fonts, formats, and layouts of the provided content. It was developed by World Wide Web Consortium (W3C).

JavaScript

JavaScript is a programming language that enables you to control the behavior of webpages. Such as updates, animated graphics, or any complex features you wish to implement.
